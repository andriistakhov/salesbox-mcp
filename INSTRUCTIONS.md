# SalesBox MCP — інструкція користувача

MCP-сервер, що дає Claude доступ до **SalesBox API** (кешбеки, категорії, чати, знижки, SEO, кастомні поля, фільтри) прямо з діалогу.

- **Ендпоінт:** `https://mcp.salesbox.ua/mcp`
- **Health-перевірка:** `https://mcp.salesbox.ua/health` → має вернути `{"ok":true,...}`

---

## 1. Що потрібно

Лише **один токен** — твій особистий **SalesBox-токен**. Під ним виконуються всі запити до твоєї компанії.

Отримати: [SalesBox support-бот](https://t.me/salesboxsupport_bot).

> Кожна компанія працює під **своїм** токеном. Один токен = одна компанія.

---

## 2. Підключення

### Claude Code

Додай у `~/.claude.json` (або `.mcp.json` у проєкті):

```json
{
  "mcpServers": {
    "salesbox": {
      "type": "http",
      "url": "https://mcp.salesbox.ua/mcp",
      "headers": {
        "Authorization": "Bearer <ТВІЙ_SALESBOX_ТОКЕН>"
      }
    }
  }
}
```

Або однією командою:
```bash
claude mcp add salesbox --transport http https://mcp.salesbox.ua/mcp \
  --header "Authorization: Bearer <ТВІЙ_SALESBOX_ТОКЕН>"
```

### Claude Desktop

У `claude_desktop_config.json` — та сама секція `mcpServers`, що вище. Після збереження перезапусти Claude Desktop.

### Перевірка

Після підключення напиши Claude: **«Покажи список кешбеків у SalesBox»** — має повернути дані твоєї компанії.

---

## 3. Що вміє (інструменти)

| Група | Дії |
|---|---|
| **Кешбеки** | список, отримати, створити, оновити, видалити |
| **Категорії** | список, отримати, масове створення/оновлення, видалення, порядок |
| **Категорії — кастомні поля** | отримати, задати, очистити |
| **Категорії — SEO** | отримати/задати/очистити, масові операції, список з пагінацією |
| **Чати** | чати користувача (активні/архівні), надіслати повідомлення |
| **Компанія** | кастомні поля та SEO (отримати/задати/очистити) |
| **Кастомні поля (визначення)** | список, створити, оновити, видалити |
| **Знижки** | список, отримати, створити, оновити, видалити (по одній / масово) |
| **Фільтри** | список, масове створення/оновлення/видалення |
| **Універсальний** | `salesbox_raw_request` — будь-який `/openapi/*` ендпоінт, якщо окремого інструмента нема |

### Приклади запитів до Claude

- «Створи кешбек **Літня акція**, тип `CATEGORY_CASHBACK`, розмір `10`.»
- «Покажи знижку з id `f862ead3-...` українською.»
- «Онови SEO-title категорії `shoes` на *"Взуття 2026"*.»
- «Знайди активні чати користувача `ac3b1588-...`.»
- «Через `salesbox_raw_request` витягни список користувачів, сторінка 1.»

---

## 4. Обмеження (rate limits)

- Усі `/openapi/*` — **100 запитів/хв** на компанію.
- Завантаження фото — додатково **200/год** на компанію.
- При перевищенні API віддає **HTTP 429**; сервер **автоматично чекає** `Retry-After` і повторює запит. Спеціально нічого робити не треба.

---

## 5. Якщо щось не працює

| Симптом | Причина / рішення |
|---|---|
| Помилка `UNAUTHORIZED` у відповіді інструмента | Невірний або відсутній SalesBox-токен. Перевір заголовок `Authorization: Bearer ...` або перевипусти токен через бот. |
| Claude не бачить сервер | Перевір `https://mcp.salesbox.ua/health` у браузері; перезапусти клієнт після зміни конфіга. |
| Повільно / затримки | Спрацював ліміт 100/хв — зачекай хвилину. |

Швидка перевірка ендпоінта з терміналу:
```bash
curl -s https://mcp.salesbox.ua/health
```
